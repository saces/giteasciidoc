==== Configure gitea

Add this to app.ini

----
[markup.asciidoc]
ENABLED         = true
FILE_EXTENSIONS = .adoc,.asciidoc
RENDER_COMMAND  = /scripts/giteawrapper.sh
IS_INPUT_FILE   = false
----
